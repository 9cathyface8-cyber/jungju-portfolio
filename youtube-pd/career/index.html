유튜브 콘텐츠 PD

스포츠 기반 YouTube 10만 채널 운영, CJ ENM 산하 DIA TV 파트너 6년 활동, FIFA22 광고 캠페인 연계, Instagram Reels 3주 만에 약 2.2만 팔로워 성장을 경험했습니다.

저는 아이디어 도출에서 시나리오와 스토리보드, 촬영 흐름, 편집 리듬, 제목·썸네일, 업로드 후 데이터 개선까지 A to Z를 직접 수행해온 PD입니다. “감자 하나로 강원도에서 서울까지 갈 수 있을까?” 콘텐츠에서는 배달 수행, 일별 수수료, 미션 보너스를 미션형 서사로 풀어내며 생활 플랫폼과 라이더 경험을 시청자가 이해하기 쉬운 콘텐츠 맥락으로 다뤘습니다.

PD SUMMARY

영상 제작 A to Z를
반복 가능한 운영 흐름으로 남깁니다

유튜브 채널에서 중요한 것은 한 번의 아이디어가 아니라, 시청자가 클릭하고 계속 보게 만드는 흐름을 반복적으로 설계하는 일이라고 봅니다. 저는 제목·썸네일·첫 장면·구성·편집 리듬·댓글 반응·시청 데이터까지 한 덩어리로 보고 다음 콘텐츠로 되돌리는 방식으로 채널을 운영해왔습니다.

Planning
소재를 영상 구조로 바꿉니다
타깃이 이미 궁금해하는 것, 댓글에서 반복되는 질문, 팬덤의 감정 언어, 현장에서 직접 겪은 사건을 영상의 질문과 시나리오로 바꿉니다.
Production
촬영과 편집까지 연결합니다
기획서에서 멈추지 않고 촬영 흐름, 화면 전환, 자막 리듬, 썸네일 방향까지 제작물의 형태로 구현합니다.
Data Loop
반응을 다음 편으로 돌립니다
조회수, 시청 지속 시간, CTR, 댓글 반응을 확인해 제목·썸네일·오프닝·구성을 다음 콘텐츠에서 다시 조정합니다.
SELECTED CASES

유튜브 PD 직무와 연결되는
핵심 제작 사례

단순히 조회수가 높았던 콘텐츠가 아니라, 어떤 판단으로 기획하고 어떤 방식으로 시청 흐름을 만들었는지, 그리고 그 경험을 채널 운영에 어떻게 재사용할 수 있는지를 기준으로 정리했습니다.

우피터 · 스포츠 팬덤 YouTube 채널
YouTube / Football / Channel Growth
How : 사람들이 자국 선수에게 몰입하고 응원한다는 상수에 손흥민 선수라는 대중적 변수를 결합했습니다. 축구 뉴스와 팬덤 심리를 빠르게 해석해 제목, 썸네일, 오프닝, 영상 흐름에 반영했습니다.
Result
우피터 채널로 10만 구독자 YouTube 운영 경험을 만들었고, CJ ENM 산하 DIA TV 파트너 크리에이터 네트워크에서 6년 활동했습니다. 2022년 한국 스포츠/엔터테인먼트 분야 종합지표 6위를 달성했습니다. (출처 : Nox influencer)
PD Value
팬덤이 이미 쓰는 언어를 읽고, 시청자가 클릭할 이유와 계속 볼 이유를 영상 구조 안에 배치하는 채널 운영 경험입니다.
10만 구독자 6년 DIA TV 스포츠/엔터 한국 6위
FIFA22 X WPT · 브랜드 협업 콘텐츠
Brand Campaign / YouTube Integration
How : 광고를 별도 삽입물처럼 보이게 하기보다, 시청자가 이미 보고 싶어 하는 축구·게임 장면 안에 자연스럽게 배치했습니다. FIFA 시리즈의 익숙한 비주얼 문법과 손흥민 중심 후킹을 활용했습니다.
Result
스포츠 YouTube 운영으로 확보한 팬덤 이해와 채널 신뢰를 바탕으로 FIFA22 X WPT 광고 콘텐츠를 집행했습니다. 광고 메시지를 콘텐츠 흐름 안에 배치해 시청자가 피하지 않는 브랜드 노출을 만들었습니다.
PD Value
협업 목적, 시청자 반응, 브랜드 노출의 균형을 맞추며 광고를 영상 서사의 일부로 설계한 경험입니다.
FIFA22 광고 캠페인 댓글 반응 기반 검증 게임플레이 맥락형 노출
밖정주 · 자전적 실험형 YouTube
YouTube / Mission Story / Field Content
How : 개인의 실패, 회복, 몸의 변화, 돈을 만들어내는 과정을 청년들이 자기 삶에 대입할 수 있는 실험형 서사로 바꿨습니다. “감자 하나로 강원도에서 서울까지 갈 수 있을까?” 콘텐츠에서는 이동 미션 안에 배달 수행과 보상 구조를 넣어 현장형 콘텐츠로 구성했습니다.
Result
공개 페이지 기준, “3달 안에 35KG...” 13만회, “인생이 최악의 상황...” 9.5만회, “72시간 안에 감자 한 개를 팔아서 얼마까지 만들 수 있을까?” 9.1천회 등 자전적 실험형 콘텐츠가 2030 청년층의 공감 지점을 공략했습니다.
PD Value
수행형 콘텐츠에서 중요한 미션, 제약, 시간 압박, 보상 구조를 시청자가 이해하기 쉬운 영상 흐름으로 바꾼 경험입니다.
13만회 감량 콘텐츠 9.5만회 인생 실험 배달 미션형 서사
냥정주 · 숏폼 후킹과 시리즈 설계
Instagram Reels / Short-form / Series
How : 반려동물이라는 보편적 상수에 “모르는 고양이가 들어왔다”, “김치통에서 자다 걸렸다”, “고양이도 고소할 수 있나요?” 같은 특수 상황 변수를 결합했습니다. 첫 장면만으로 질문이 생기도록 후킹을 설계했습니다.
Result
약 3주 만에 Instagram 팔로워 약 2.2만 명 성장. 최고 조회수 664만회를 포함해 249만회, 212만회, 178만회, 133만회, 117만회, 101만회, 100만회 등 산고양이 시리즈 대표 콘텐츠가 반복적으로 고조회수를 만들었습니다.
PD Value
숏폼의 첫 1초, 자막 리듬, 반복 가능한 시리즈 구조를 설계하고 긴 호흡의 YouTube 콘텐츠 소재 발굴에도 연결할 수 있는 경험입니다.
최고 664만회 3주 약 2.2만 성장 시리즈 포맷 검증
고양이 콘텐츠 관심을 선예약 응답으로 전환
Audience Conversion / Reservation Demand
How : 산고양이들이 등장하는 촬영 배경지가 실제 펜션 사유지라는 맥락을 활용했습니다. 펜션 오픈 과정에서 콘텐츠로 형성된 호기심과 애착을 선예약 폼으로 연결해, 시청자 반응이 실제 수요로 이어지는지 확인했습니다.
Result
Google Forms 응답 요약 기준, 2일 만에 선예약 응답 234개를 확보했습니다. 예약 의향군 89.3%, 주요 이용 인원 2~4인 92.7%, 1박 선호 72.7% 등 상품 구성 판단에 필요한 수요 데이터도 함께 확인했습니다.
PD Value
조회수와 팔로워를 끝점으로 두지 않고, 영상이 만든 관심을 예약 문의, 고객 니즈, 상품 기획 데이터로 연결한 경험입니다.

PD WORKFLOW

아이디어를 영상으로 만들고
반응을 다음 기획으로 되돌립니다

유튜브 콘텐츠는 기획, 현장 판단, 편집 감각, 데이터 개선이 끊기지 않을 때 강해진다고 봅니다. 저는 이 흐름을 한 번의 제작물이 아니라 반복 가능한 제작 루프로 관리합니다.

01
아이디어 도출
타깃, 팬덤 언어, 댓글 질문, 현장 경험에서 소재를 뽑고 한 문장의 영상 질문으로 정리합니다.
02
시나리오·스토리보드
도입 후킹, 중간 전환점, 클라이맥스, CTA를 기준으로 시청 흐름을 먼저 설계합니다.
03
현장 연출
예상과 다른 상황이 생겨도 미션, 리액션, 설명 장면을 살려 영상의 리듬을 유지합니다.
04
편집·자막·썸네일
컷 전환, 자막 밀도, 정보 노출 순서, 제목·썸네일 카피를 함께 조정합니다.
05
업로드 최적화
타이틀, 썸네일, 키워드, 업로드 후 초기 반응을 보며 시청자의 클릭 이유를 점검합니다.
06
데이터 피드백
조회수, 시청 지속 시간, CTR, 댓글 반응을 다음 소재와 구성 개선으로 연결합니다.
