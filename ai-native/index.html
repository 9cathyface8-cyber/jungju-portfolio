01 · TREND RESEARCH PAUSED — DATA/API COST

Viral Radar

최신 트렌드를 탐색하기 위해 만든 트렌드 레이더입니다. 이미 유행한 결과가 아니라, 마케터가 최신 트렌드 캐치를 할 수 있을 만큼 이른 초기 확산 신호를 찾는 것이 목표였습니다.

후보 수집 → 반복 스냅샷 → 성장 속도·가속도·계정 기준선 비교 → 정량 후보 선별 → LLM 해석 → 사람이 원본 확인
기존 병목

여러 플랫폼을 직접 훑으면 이미 커진 콘텐츠와 초기 신호를 분리하기 어렵고 탐색 시간이 길어집니다.

AI가 맡은 판단

정량 모델이 후보를 먼저 선별하고, LLM은 훅·포맷·확산 이유를 사람이 이해할 언어로 해석합니다.

사람의 검토

마케터가 영상 원본, 출처와 근거 지표를 함께 확인하고 실제 제작에 참고할지 결정합니다.

검증 결과

외부 접속 가능한 UI와 분석 흐름을 구현했습니다. 다수 플랫폼의 반복 수집 비용이 커 상시 운영은 보류했습니다.

확인한 한계

AI 해석보다 신뢰할 모집단, 시간축 데이터와 플랫폼별 데이터 품질이 제품 성능을 먼저 결정했습니다.

자연어 문의 → 날짜·인원·객실·금액·요청사항 구조화 → 가능 여부 확인 → 예외 Human Handoff → 예약 확정 정보 발송

AI가 맡은 판단

문의에서 예약 조건을 추출하고 확인해야 할 정보와 누락된 항목을 구조화합니다.

사람의 검토
